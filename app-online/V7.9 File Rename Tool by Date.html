ប្តូរឈ្មោះឯកសារទាំងអស់ក្នុងថត ទៅជាលេខរៀងតាមលំដាប់កាលបរិច្ឆេទ
(ឯកសារដែលចាស់ជាងគេ ឬដាក់ចូលថតមុនគេ នឹងក្លាយជាលេខ ១)

✅ របៀបកែក្នុងថតផ្ទាល់ (Native) 📦 របៀបទាញយកជា ZIP (Fallback)
💡 របៀបប្រើ៖ ជ្រើសរើសថតឯកសារ → ពិនិត្យមើលលទ្ធផល → ចុច "ចាប់ផ្តើមប្តូរឈ្មោះ"។ ឯកសារនឹងត្រូវបានកែឈ្មោះដោយផ្ទាល់ក្នុងថត។ សូមបម្រុងទុកឯកសារសំខាន់ៗជាមុន!
💡 របៀបប្រើ៖ ជ្រើសរើសឯកសារពីថត → ពិនិត្យមើលលទ្ធផល → ទាញយកជា ZIP។ ឯកសារដើមនឹង មិនត្រូវបានកែប្រែ ទេ។ សាកសមសម្រាប់ browser ដែលមិនគាំទ្រ File System Access API។
