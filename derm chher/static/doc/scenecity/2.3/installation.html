Installating, updating, removing the addon#

Installing and enabling the addon#

Installation is just like any other Blender addon, there are no special steps:

  1. Download the addon zip file from Gumroad

  2. Go to Blender’s user preferences => Addons

  3. Click the “Install” button

  4. Select the addon’s zip file you downloaded. After this step the addon is installed but not enabled yet.

  5. Find SceneCity 2 (you can use the search field). Enable it.

Setting the addon preferences#

You have to specify the complete path to your Megascans root folder. For exemple on Windows something like C:\Users\YourUserName\Documents\Megascans Library. SceneCity will later read from this folder, but will never modify it.

Removing the addon#

Removing the addon is just like any other Blender addon, there are no special steps:

  1. Go to Blender’s user preferences => Addons

  2. Find SceneCity 2 (you can use the search field)

  3. Expand the addon and click on its “Remove” button. Good bye :(

Updating the addon#

Bugfix releases#

When you want to update to the next bugfix version of SceneCity, for exemple 2.0.0 to 2.0.1, it’s very simple: remove the addon, then install the newer version by following the steps on this page.

Minor releases#

When you want to update to the next minor version of SceneCity, for exemple 2.0.x to 2.1.x, it’s a bit less simple:

  • remove the addon by following the steps on this page

  • then install the newer version by following the steps on the newer version’s doc page, as it may be different from this one
